# E-Commerce Product Catalog

## Name

Darshana Selvakumar

## Project Idea

This project is an E-Commerce Product Catalog web application similar to Amazon.

The application will allow users to:

* View products
* Search products
* Filter products by category
* View product details
* Add products to cart
* Manage products through an admin panel

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* PostgreSQL

## Future Features

* User authentication
* Online payment integration
* Product reviews and ratings
* Order tracking
